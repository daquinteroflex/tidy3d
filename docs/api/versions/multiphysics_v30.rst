Multiphysics 3.0 API Migration
-------------------------------

This page aims to illustrate and document important API migration changes from 2.0 to 3.0

Motivation
~~~~~~~~~~

As ``tidy3d`` has grown, we have added new solvers of different physics.
Ultimately, the goal of any simulation software is to represent a physical scenario as accurately as possible.
Part of the ``tidy3d`` ethos is to make the user experience as simple and straightforward as possible.
